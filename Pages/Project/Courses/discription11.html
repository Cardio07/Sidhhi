HARDWARE

Course Overview

The mixture of physical elements or pieces that make up the
operating machine is Computer Hardware. It includes a display,
keyboard, mouse, hard disc drive, RAM, motherboard, graphics
card, etc. Computer hardware is an important area of computer
science. Computer hardware course is mandatory for candidates
pursuing application employment in this field. In this sector,
there are many short and long-term courses which can be taken
after passing 12th class.

This Course Comprises of:

1.Microprocessors.
2.RAM,SMPS.
3.ADD-ON Cards.
4.Monitors.
5.Operating System.
6.Storage Devices,Viruses.
7.Trouble-Shooting On Computer.
