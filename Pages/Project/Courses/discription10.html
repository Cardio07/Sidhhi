AUTO CAD

Course Overview

AutoCAD is a computer-aided design software developed by the
company Autodesk (hence the name AutoCAD). It allows you to
draw and edit digital 2D and 3D designs more quickly and easily
than you could by hand.

This Course Comprises of:

1.Computer Concept & Fendamental.
2.Operating System and Introduction.
3.Customizing Auto Cad, Using Tools.
4.Creating Drawing Using Wizard, 2D and 3D.
